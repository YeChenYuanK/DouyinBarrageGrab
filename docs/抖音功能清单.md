# 抖音功能清单（基准：e12ddbe0e2ba706f5651d9e3ed89c7585b46404e）

## 1) 连接与订阅

- WS 识别规则：`webcast\d+-ws-web-\w+\.(douyin|amemv)\.com`
- 命中后订阅：`DataReceived += WebSocket_DataReceived`
- 轮询降级入口：`/webcast/im/fetch` + `application/protobuffer`
- 轮询包透传解析层：`FireOnFetchResponse(HttpResponseEventArgs)`

## 2) 抖音协议解析

- WS 外层：`WssResponse` 反序列化
- 压缩校验：`compress_type=gzip`
- 内层解析：gzip 解压 `Payload` -> `Response`
- HTTP 轮询解析：HTTP payload -> `Response`
- 统一分发入口：`DoMessage(Message msg, string processName)`

## 3) 去重机制

- 去重键：`msg.Method + msg.msgId`
- 缓冲策略：每种 `Method` 保留最近 300 条 `msgId`
- 重复消息直接丢弃，不二次触发事件

## 4) 消息类型支持（8类）

- `WebcastChatMessage` -> `OnChatMessage`
- `WebcastLikeMessage` -> `OnLikeMessage`
- `WebcastGiftMessage` -> `OnGiftMessage`
- `WebcastMemberMessage` -> `OnMemberMessage`
- `WebcastSocialMessage` -> `OnSocialMessage`
- `WebcastRoomUserSeqMessage` -> `OnRoomUserSeqMessage`
- `WebcastControlMessage` -> `OnControlMessage`
- `WebcastFansclubMessage` -> `OnFansclubMessage`

## 5) 配置能力

- 进程过滤：`app.filtering.processFilter`
- 域名过滤：`app.filtering.hostNameEnabled/hostNameList`
- 弹幕打印过滤：`app.barrage.printFilter`
- 弹幕推送过滤：`app.barrage.pushFilter`
- 弹幕日志过滤：`app.barrage.logFilter`
- 强制轮询：`app.barrage.polling.enabled`
- 轮询间隔：`app.barrage.polling.interval`

## 6) 服务启动与输出

- 启动项：`AppRuntime.WsServer.StartListen()`
- 对外能力：启动代理与弹幕 WS 推送服务
- 启动提示：输出 `ServerLocation`（供游戏端连接）

---

说明：本清单仅包含抖音能力，不包含任何快手扩展项。
